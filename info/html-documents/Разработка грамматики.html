Разработка грамматики

Определим грамматику логических выражений языка C++ G[‹Expr›] в нотации Хомского с продукциями P:

  1. ‹Expr› → ‹AndExpr› {||‹AndExpr›}
  2. ‹AndExpr› → ‹NotExpr› {&&‹NotExpr›}
  3. ‹NotExpr› → !‹SimpleExpr› | ‹SimpleExpr›
  4. ‹SimpleExpr› → ‹Identifier› | ‹BooleanLiteral› | (‹Expr›) | ‹ComparisonExpr›
  5. ‹ComparisonExpr› → ‹ArithmeticExpr› ‹RelationalOp› ‹ArithmeticExpr›
  6. ‹ArithmeticExpr› → ‹Term› {‹AdditiveOp›‹Term›}
  7. ‹Term› → ‹Factor› {‹MultiplicativeOp›‹Factor›}
  8. ‹Factor› → (‹ArithmeticExpr›) | ‹Identifier› | ‹IntegerLiteral› | ‹FloatingPointLiteral›
  9. ‹RelationalOp› → < | > | <= | >= | == | !=
  10. ‹AdditiveOp› → + | -
  11. ‹MultiplicativeOp› → * | /

Следуя введенному формальному определению грамматики, представим G[‹Expr›] ее составляющими:

  • Z = ‹Expr›;
  • VT = {<, >, ==, !=, +, -, *, /, !, &&, ||, (, ), true, false, 0, 1, 2, ..., 9, a, b, c, ..., z, A, B, C, ..., Z};
  • VN = {‹Expr›, ‹AndExpr›, ‹NotExpr›, ‹SimpleExpr›, ‹ComparisonExpr›, ‹ArithmeticExpr›, ‹Term›, ‹Factor›, ‹RelationalOp›, ‹AdditiveOp›, ‹MultiplicativeOp›}.
